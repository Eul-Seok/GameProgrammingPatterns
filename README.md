# GameProgrammingPatterns

게임 개발에서는 장기적인 개발 속도(좋은 아키텍처), 게임 실행 속도(성능), 그리고 단기적인 개발 속도(빠른 기능 구현) 이 세 가지 목표를 균형있게 유지하는게 중요하다.

디자인 패턴에 너무 치중하게 되면, 성능이나 단기적인 개발 속도가 떨어질 수 있다.
반면에, 단기적인 개발에 치중한다면 유지보수 측면에서 문제가 생길 수 있다.
